# 🌤 Weather App (Android)

An Android app that shows real-time weather using OpenWeather API.

---

## ✨ Features
- 🔍 Search weather by city
- 🌡 Displays temperature
- 💧 Shows humidity
- ☁ Shows weather condition
- 🎨 Clean UI with background

---

## 🛠 Tech Stack
- Java
- XML (Android UI)
- OpenWeather API

## 🚀 How to Run
1. Clone the repository
2. Open in Android Studio
3. Add API key in `local.properties`:
   WEATHER_API_KEY=YOUR_API_KEY
4. Run the app
